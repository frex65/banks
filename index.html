ReadMe for code challenge day 6

Installation

  1. Plonk this folder in your server's docs directory.
  2. Browse to the index.html file to run the tests.

API

The only API call is to reallocate.php. Would omit the .php on live system using .htaccess or OS alternative.

reallocate.php accepts a JSON string in a format similar to this:

{"banks":["2","3","4","5","..."]}

Test scenarios

If you point your browser at the runTests.php page, it will display a set of test scenarios and run them through the API before showing you the result. You can add your own tests by appending them to the testData array at the top of the app/app.js file.

Dependencies

Only jQuery, a copy of which is in the assets directory.
